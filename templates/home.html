  • $select:

    Occasionally you don't want to return all of the data available for a resource. You can use the $select option to limit the fields that are returned for a request.

  • $filter:

    Filtering properties by field values can be accomplished using the $filter option. All of the names in the $filter option are case-sensitive and match the name of the fields provided by the property resource.

    options:
    • eq : equals
    • lt : less than
    • le : less than or equals
    • gt : greater than
    • ne : not equals
    • and
    • or
    • not
  • $orderby:

    You can sort listings using the $orderby option. Just specify the field and the direction of the sort.

  • $top:

    Designates the maximum number of matching items returned.

  • $skip:

    Designates the number of matching items to omit before returning any items.
